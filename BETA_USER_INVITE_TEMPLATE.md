## Hướng Dẫn Đăng Ký

1. Mở link staging.
2. Chọn Đăng ký.
3. Tạo tài khoản bằng email test.
4. Hoàn tất onboarding.
5. Vào Dashboard.

## Checklist Test Nhanh

1. Tạo một lead thủ công.
2. Thêm một note.
3. Tạo một reminder/follow-up.
4. Mở pipeline và đổi trạng thái lead.
5. Thử import file CSV mẫu nếu có.
6. Gửi feedback trong app.
